# 使用示例

本文档给出各功能模块的典型操作步骤与输入输出示例。

**说明**：除 ETF 轮动外，下述回测（量化回测、因子挖掘回测、选股池组合回测）**均未考虑滑点与手续费**，结果仅供策略参考，实盘需自行考虑交易成本。详见 [常见问题 - 回测是否考虑滑点与手续费](常见问题.md#回测是否考虑滑点与手续费)。

## 量化交易面板

### AI 实时决策

- **输入**：股票代码（如 600519.SH）；选择是否有持仓，如有则输入开仓价格和持仓天数。
- **操作**：点击「🤖 获取 AI 决策」。
- **输出**：实时技术指标分析（MA、RSI、MACD、布林带等）；AI 决策（买入/卖出/观望 + 信心度）；决策理由；止损价、止盈价、建议仓位等风险管理建议。

### AI Agent 量化

- **输入**：自然语言，例如「请对贵州茅台进行量化分析，回测多因子策略」。
- **操作**：点击「🤖 运行 AI 量化分析」。
- **输出**：完整 AI 驱动流程（数据收集 → 策略回测 → AI 审计 → 风险评估与报告）。

### 普通回测

- **输入**：股票代码（如 600519.SH）；选择策略类型、初始资金。
- **操作**：点击「📊 开始回测」。
- **输出**：回测结果（收益率、夏普、回撤、胜率等）+ 7 合 1 可视化图表与买卖点标注。

## 因子挖掘（因子挖掘页）

### 深度挖掘 + 回测

1. 配置股票池（如指数 000300 或分行业龙头）。
2. 设置因子模式（多因子）、迭代次数、TopN、调仓周期、基准代码等；可勾选「使用 Agent 工作流」。
3. 点击「开始深度搜索」→ 编排/评价/审查 Agent 依次执行，得到最佳因子组合与审查报告。
4. 在回测参数区设置 TopN、调仓周期、个股权重（等权/按得分加权/凯利）；可选勾选「稳健性检查」。
5. 点击「执行回测」→ 查看总收益、夏普、回撤、Alpha/Beta、收益曲线与（若勾选）稳健性检查结果表。
6. 可填写摘要标题并「保存摘要」，便于在「已保存的回测摘要」中查看。

详见 [因子挖掘与回测](因子挖掘与回测.md)。

## 研究与新闻（右侧面板）

### Research Agent

- **输入**：例如「查询贵州茅台 2023 年年报的营收和净利润情况」。
- **操作**：调用 Research Agent。
- **输出**：财务数据分析；📄 参考文档（CNINFO 公告列表、知识库搜索结果、行情数据来源）。

### News Agent

- **输入**：例如「获取贵州茅台最近的新闻」。
- **操作**：调用 News Agent。
- **输出**：市场与个股新闻（当前 News Agent 暂不可用，恢复时间另行说明）。

详见 [研究资讯前端使用](研究资讯前端使用.md)。

## ETF 轮动模拟盘

### 创建模拟账户

1. 切换到「ETF 轮动」标签页。
2. 选择「模拟盘」模式。
3. 输入账户 ID 和初始资金。
4. 点击「创建账户」。

### 查看账户信息

- 总资产、可用资金、持仓市值。
- 盈亏情况（金额和百分比）。
- 当前持仓详情（ETF 代码、股数、成本价、现价、盈亏）。
- 最近交易记录（买入/卖出、价格、原因）。

### 执行自动调仓

1. 配置 ETF 池（添加要轮动的 ETF 代码）。
2. 设置策略参数（Top-K、回看天数等）。
3. 点击「执行自动调仓」，系统会根据调仓建议自动执行买卖操作。

### 查看调仓建议

- 点击「查看调仓建议」查看当前建议持有的 ETF。
- 显示所有 ETF 的得分排名；建议持有 Top-K 只 ETF。

### 实盘交易（需配置 QMT）

- 实盘需连接 QMT 客户端。确保 QMT 已启动并登录后，通过实盘交易 API 执行下单；系统通过 QMT SDK 与券商交互。使用前请在模拟盘充分验证策略。

更多说明见 [ETF轮动前端与模拟盘](ETF轮动前端与模拟盘.md)。
